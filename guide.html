※ Page

파일명 네이밍

- 소문자만 사용하며, 대문자는 사용하지 않는다.
- 네이밍이 길경우에는 언더바("_")를 사용한다.
- 팝업페이지는 "pop_팝업명"으로 네이밍한다.

CSS 클래스와 변수 네이밍

- 소문자만 사용하며, 대문자는 사용하지 않는다.
- 네이밍이 길경우에는 하이픈("-")을 사용한다.

이미지 네이밍

- 소문자만 사용하며, 대문자는 사용하지 않는다.
- 네이밍이 길경우에는 언더바("_")를 사용한다.
로고 : logo_
폼 : form_
아이콘 : icon_
버튼 : btn_
화살표 : arrow_
플러그인 : 플러그인명_
공통 헤더에 있는 이미지 : conthead_
공통 메뉴에 있는 이미지 : lnb_

Page 작업

- #wrap에 서비스별 해당 페이지에 맞는 클래스를 선언한다.

1. ENS인 경우
<div id="wrap" class="ems">

2. RNS인 경우
<div id="wrap" class="rns">

3. SMS인 경우
<div id="wrap" class="sms">

4. PUSH인 경우
<div id="wrap" class="push">

5. 공통설정인 경우
<div id="wrap" class="sys">
